NOTE: These instructions will work only for DevC++/GCC compiler only.

Please follow the instructions below to add the Linker Code, in order to execute Sound Files. Otherwise compile time error will occur.

1) goto 'tools' and select 'compiler options'
2) open 'general tab'
3) add the below linker code in 2nd box named as 'add the following commands when calling the linker'
4) if any other commands are already added then, give few spaces and then add the likner code by simply copy/pasting from below.

LINKER CODE:     -lwinmm
