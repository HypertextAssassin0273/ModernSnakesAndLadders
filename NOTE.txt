Please follow the instructions below to add the Linker Code, in order to execute Sound Files. Otherwise Compile Time Error will occur.

LINKER CODE:     -lwinmm

Instructions:
1) goto 'tools' and select 'compiler options'
2) open 'general tab'
3) add the above linker code in 2nd box named as 'add the following commands when calling the linker'
4) if any other commands are already added then, give few spaces and then add the likner code by simply copy/pasting from above.
